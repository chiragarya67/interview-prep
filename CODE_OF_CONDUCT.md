# Code of Conduct

## The Short Version

Be respectful. Be helpful. Don't be a jerk.

---

## The Longer Version

**We welcome everyone** regardless of experience level, background, gender, nationality, or anything else. Whether you're a senior dev or opening your first ever PR — you're welcome here.

**We expect contributors to:**
- Be kind and patient, especially with beginners
- Give constructive feedback on PRs, not personal criticism
- Accept feedback on your own PRs without getting defensive
- Ask questions openly — no question is too basic

**We don't tolerate:**
- Harassment, insults, or discrimination of any kind
- Dismissing someone for being a beginner
- Spam PRs or low-effort contributions just to farm GitHub stats

---

## Reporting Issues

If someone is violating this code of conduct, reach out via GitHub Discussions or open a private issue. We'll handle it.
